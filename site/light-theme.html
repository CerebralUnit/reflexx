Intelligent End-to-End Network Solutions

Application-driven, service-oriented architectures (SOA) and virtualization have banished the client-server model from the data center. Cloud computing, too, makes heavy use of server virtualization, which reshapes data center traffic flows and increases bandwidth demands at the server edge. By 2014, network planners should expect more than 80 percent of traffic in the data center's local area network (LAN) to be between servers.

These efforts at flexibility can be hampered by legacy data center networks. They cannot provide high enough bandwidth and low enough latency between server connections to support highly mobile virtual workloads.
